## 1. Bienvenue sur Mini CRM

### Qu'est-ce que Mini CRM ?

Mini CRM est une application web simple et intuitive qui vous permet de **gérer vos contacts professionnels** de manière efficace. Que vous soyez freelance, entrepreneur ou membre d'une petite structure, Mini CRM vous aide à :

- **Centraliser** tous vos contacts au même endroit
- **Retrouver** rapidement une personne ou une entreprise
- **Planifier** vos relances commerciales
- **Marquer** vos contacts importants comme favoris
- **Sécuriser** vos données (chaque utilisateur voit uniquement ses propres contacts)

### À qui s'adresse Mini CRM ?

- **Freelances** qui gèrent leurs clients et prospects
- **Entrepreneurs** qui développent leur réseau
- **Commerciaux** qui suivent leurs contacts
- **Consultants** qui organisent leurs relations professionnelles

## 2. 🚀 Premiers pas

### 2.1 Créer votre compte

#### Étape 1 : Accéder à la page d'inscription

1. Ouvrez votre navigateur web (Chrome, Firefox, Edge, Safari)
2. Rendez-vous sur la page d'accueil de Mini CRM
3. Cliquez sur **"Inscription"** dans le menu

#### Étape 2 : Remplir le formulaire d'inscription

Vous devrez fournir les informations suivantes :

| Champ | Description | Exemple |
|-------|-------------|---------|
| **Nom d'utilisateur** | Identifiant unique (3-20 caractères) | `juklau` |
| **Email** | Votre adresse email | `votre.email@example.com` |
| **Mot de passe** | Minimum 8 caractères, 1 majuscule, 1 chiffre | `MonMotDePasse123` |
| **Confirmation** | Retapez votre mot de passe | `MonMotDePasse123` |

#### Étape 3 : Validation

1. Cliquez sur le bouton **"S'inscrire"**
2. Un message de confirmation s'affiche
3. Vous êtes automatiquement redirigé vers la page de connexion

---

### 2.2 Se connecter

#### Connexion simple

1. Rendez-vous sur la page **"Connexion"**
2. Saisissez votre **nom d'utilisateur**
3. Saisissez votre **mot de passe**
4. Cliquez sur **"Se connecter"**

**Astuce** : Utilisez le bouton 👁️ (œil) pour afficher/masquer votre mot de passe pendant la saisie.

#### Que faire si j'ai oublié mon mot de passe ?

Pour le moment, Mini CRM ne dispose pas de fonction de réinitialisation automatique. Contactez votre administrateur système pour réinitialiser votre mot de passe.

## 3. Gestion de votre compte

### 3.1 Voir vos informations

Votre **nom d'utilisateur** s'affiche en haut à gauche du menu de navigation quand vous êtes connecté.

### 3.2 Se déconnecter

Pour vous déconnecter en toute sécurité :

1. Cliquez sur le bouton **"Déconnexion"** dans le menu
2. Vous êtes automatiquement redirigé vers la page de connexion
3. Vos données restent sécurisées

**Important** : Déconnectez-vous toujours si vous utilisez un ordinateur partagé.

## 4. Gestion des contacts

### 4.1 Ajouter un nouveau contact

#### Accéder au formulaire d'ajout

1. Cliquez sur **"Ajouter un contact"** dans le menu
2. Vous accédez au formulaire d'ajout

#### Remplir les informations

| Champ | Requis ? | Description |
|-------|------|-------------|
| **Nom** |  Oui | Nom de famille du contact |
| **Prénom** |  Oui | Prénom du contact |
| **Entreprise** |  Oui | Nom de l'entreprise |
| **Email** |  Oui | Adresse email professionnelle |
| **Téléphone** |  Oui | Numéro de téléphone (10-15 chiffres) |
| **Type de contact** |  Oui | Client / Prospect / Partenaire / Fournisseur |
| **Date de relance** |  Non | Date prévue pour la prochaine relance |
| **Statut de relance** |  Oui | À relancer / En cours / Terminé |
| **Note** |  Non | Notes personnelles (max 500 caractères) |
| **Photo** |  Non | Photo de profil du contact |

#### Ajouter une photo de profil

1. Cliquez sur l'icône de photo
2. Cliquez sur **"Télécharger"**
3. Sélectionnez une image sur votre ordinateur (JPG, PNG)
4. L'image s'affiche immédiatement

**Conseil** : Utilisez une photo professionnelle de bonne qualité.

#### Marquer comme favori

Cliquez sur l'étoile ⭐ pour marquer le contact comme favori. L'étoile devient jaune/pleine.

#### Valider l'ajout

1. Vérifiez que tous les champs obligatoires sont remplis
2. Cliquez sur **"Ajouter"**
3. Confirmez en cliquant sur **"Oui"** dans la fenêtre de confirmation
4. Un message de succès s'affiche
5. Le formulaire est réinitialisé pour un nouvel ajout

---

### 4.2 Consulter un contact

#### Depuis la page d'accueil

1. Parcourez la liste alphabétique de vos contacts
2. Cliquez sur la **carte** du contact que vous souhaitez consulter
3. Une fenêtre s'ouvre avec toutes les informations détaillées

#### Informations affichées

-  Photo de profil
-  Nom complet
-  Entreprise
-  Email
-  Téléphone
-  Type de contact
-  Date de relance
-  Statut de relance
-  Notes
-  Statut favori

#### Fermer la fenêtre

Cliquez sur la **croix** ❌ en haut à droite pour fermer la fenêtre de détails.

---

### 4.3 Modifier un contact

#### Modifier les informations

1. Cliquez sur la carte du contact
2. Changez les informations souhaitées
3. Les champs sont validés en temps réel (bordures vertes = OK, rouges = erreur)
4. Cliquez sur le bouton **"Modifier"** ✏️
5. Confirmez en cliquant sur **"Oui"**
5. Un message de succès s'affiche

**Astuce** : Les modifications sont sauvegardées uniquement après confirmation.

#### Modifier la photo

1. Cliquez sur la **photo de profil** dans le formulaire de modification
2. Une fenêtre s'ouvre avec 3 options :
    - **Télécharger** : choisir une nouvelle photo
    - **Supprimer** : retirer la photo (remplacée par l'image par défaut)
    - **Fermer** : annuler

#### Annuler les modifications

Si vous changez d'avis :
1. Cliquez sur **"Annuler"** ou la croix ❌
2. Les modifications ne sont pas enregistrées
3. Vous revenez à la page d'accueil

---

### 4.4 Supprimer un contact

⚠**Attention** : Cette action est **irréversible** !

#### Procédure de suppression

1. Ouvrez les détails du contact
2. Cliquez sur le bouton **"Supprimer"**
3. Une fenêtre de confirmation s'affiche :
   > **"Êtes-vous sûr de vouloir supprimer ce contact ?"**
4. Cliquez sur **"Oui"** pour confirmer (ou **"Non"** pour annuler)
5. Le contact est définitivement supprimé
6. Vous êtes redirigé vers la page d'accueil

**Conseil** : Avant de supprimer un contact, vérifiez bien que vous n'en aurez plus besoin.

## 5. Recherche et filtres

### 5.1 Recherche rapide (page d'accueil)

#### Recherche par nom du contact ou entreprise

1. Sur la **page d'accueil**, repérez la barre de recherche en haut
2. Commencez à taper un **nom** ou une **entreprise**
3. Les résultats s'affichent **instantanément** pendant que vous tapez
4. Seuls les contacts correspondants sont affichés

**Exemples** :
- Tapez `du` → affiche "Dupont", "Durand"
- Tapez `tech` → affiche tous les contacts de "TechCorp", "TechSolutions"

**Astuce** : La recherche ne tient pas compte des majuscules/minuscules.

#### Annuler la recherche

- Effacez le contenu de la barre de recherche
- Tous vos contacts réapparaissent

---

### 5.2 Recherche avancée (page Recherche)

#### Accéder à la recherche avancée

1. Cliquez sur **"Recherche"** dans le menu
2. Vous accédez à la page de recherche multicritères

#### Utiliser les filtres

##### Filtre par Type de contact

Sélectionnez un type dans la liste déroulante :
- **Tous** (par défaut)
- Client
- Prospect
- Partenaire

##### Filtre par Statut de relance

Sélectionnez un statut :
- **Tous** (par défaut)
- À relancer
- Fait

##### Combiner les filtres

Vous pouvez combiner les deux filtres :

**Exemple** :
- Type : **Client**
- Statut : **À relancer**
- → Affiche uniquement les clients à relancer

#### Lancer la recherche

1. Sélectionnez vos critères
2. Cliquez sur **"Rechercher"**
3. Un indicateur de chargement s'affiche
4. Les résultats correspondants s'affichent

#### Réinitialiser les filtres

Pour voir à nouveau tous vos contacts :
1. Sélectionnez **"Tous"** dans chaque filtre
2. Cliquez sur **"Rechercher"**

## 6. Organisation des contacts

### 6.1 Tri alphabétique

Vos contacts sont **automatiquement organisés par ordre alphabétique** selon le nom de famille.

#### Accordéons alphabétiques

```
▼ A (3 contacts)
  - Dupont Marie
  - Durand Pierre
  - Alain Sophie

▼ B (1 contact)
  - Bernard Thomas

▶ C (aucun contact)
```

- Les lettres **avec** contacts sont **visibles** et dépliables
- Les lettres **sans** contact sont **masquées**
- Le **premier accordéon** avec des contacts est **ouvert par défaut**

#### Déplier/Replier un accordéon

1. Cliquez sur la lettre (ex : **"A"**)
2. L'accordéon se déplie et affiche les contacts
3. Cliquez à nouveau pour replier

---

### 6.2 Gestion des favoris

#### Marquer un contact comme favori

**Méthode 1 : Lors de l'ajout**
- Cliquez sur l'étoile ⭐ dans le formulaire d'ajout
- L'étoile devient **jaune/pleine** ⭐ = favori

**Méthode 2 : Depuis la modification**
- Ouvrez le contact
- Cliquez sur l'étoile
- Enregistrez les modifications

#### Retirer des favoris

- Cliquez à nouveau sur l'étoile ⭐
- L'étoile redevient **vide** ☆ = contact normal

**Utilité** : Les favoris vous permettent de repérer rapidement vos contacts les plus importants.

---

### 6.3 Système de relances

#### Définir une relance

1. Lors de l'ajout ou de la modification d'un contact
2. Remplissez le champ **"Date de relance"** (optionnel)
3. Choisissez un **"Statut de relance"** :
    - **À relancer** : contact à recontacter prochainement
    - **Fait** : relance effectuée

#### Suivre vos relances

Utilisez la **recherche avancée** pour filtrer :
- **Statut = "À relancer"** → voir tous les contacts à recontacter
- Consultez la date de relance de chaque contact

**Organisation** : Créez une routine pour consulter régulièrement vos "Contacts à relancer".

## 7. Fonctionnalités avancées

### 7.1 Validation en temps réel

Pendant que vous remplissez un formulaire, Mini CRM **valide automatiquement** vos entrées :

| État | Indicateur | Signification |
|------|------------|---------------|
|  **Valide** | Bordure verte | Le champ est correctement rempli |
|  **Invalide** | Bordure rouge + message d'erreur | Le champ contient une erreur |
|  **Neutre** | Bordure grise | Champ non encore rempli |

**Exemple d'erreurs courantes** :
- Email : `"Veuillez entrer un email valide"`
- Téléphone : `"Le téléphone doit contenir 10 à 15 chiffres"`
- Mot de passe : `"Le mot de passe doit contenir au moins 8 caractères, 1 majuscule et 1 chiffre"`

**Avantage** : Vous savez immédiatement si vous avez fait une erreur, sans attendre la soumission du formulaire.

---

### 7.2 Affichage des photos

#### Photo par défaut

Si un contact n'a pas de photo, une **image par défaut** s'affiche automatiquement (silhouette générique).

#### Photos personnalisées

- Les photos sont **hébergées sur Cloudinary** (service cloud sécurisé)
- Formats acceptés : **JPG, PNG**
- Taille recommandée : **max 2 MB**

## 8. Conseils et astuces

### 8.1 Bonnes pratiques

#### Organisation des contacts

- **Remplissez un maximum de champs**
    - Plus vous avez d'informations, plus c'est facile de retrouver un contact

- **Utilisez des notes**
    - Notez le contexte de la rencontre, les sujets discutés, etc.
    - Exemple : _"Rencontré au salon de l'entrepreneuriat - intéressé par nos services de consulting"_

- **Mettez à jour régulièrement**
    - Changement de poste, de téléphone → modifiez le contact

- **Utilisez les favoris stratégiquement**
    - Marquez vos contacts VIP, vos meilleurs clients, etc.

#### Gestion des relances

- **Planifiez vos relances**
    - Définissez une date réaliste pour chaque relance

- **Vérifiez régulièrement**
    - Consultez la recherche avec filtre "À relancer" chaque semaine

**Mettez à jour le statut**
- Après avoir relancé un contact, changez le statut en "Fait"

---

### 8.2 Sécurité de vos données

#### Vos données sont isolées

- Chaque utilisateur voit **uniquement ses propres contacts**
- Même si plusieurs personnes utilisent Mini CRM, vos contacts restent **privés**

#### Recommandations de sécurité

- **Choisissez un mot de passe fort**
    - Minimum 8 caractères
    - 1 majuscule
    - 1 chiffre
    - Évitez les mots de passe évidents

- **Ne partagez jamais votre mot de passe**

- **Déconnectez-vous sur ordinateur partagé**

- **Vérifiez l'URL** avant de saisir vos identifiants

---

### 8.3 Optimisation de votre utilisation

#### Gagner du temps

- **Utilisez la recherche rapide** sur la page d'accueil pour trouver un contact en quelques secondes

- **Créez des contacts en lots** : ajoutez plusieurs contacts d'affilée après un événement réseau

- **Marquez vos favoris** pour accéder rapidement à vos contacts prioritaires

#### Éviter les doublons

Avant d'ajouter un contact :
1. Faites une recherche rapide avec son nom
2. Vérifiez qu'il n'existe pas déjà

## 9.  Dépannage

### 9.1 Problèmes de connexion

#### "Nom d'utilisateur ou mot de passe incorrect"

**Solutions** :
1.  Vérifiez votre **nom d'utilisateur** (sensible à la casse)
2.  Vérifiez votre **mot de passe** (utilisez le bouton 👁️ pour voir ce que vous tapez)
3.  Vérifiez que la touche **Caps Lock** n'est pas activée
4.  Si le problème persiste, contactez votre administrateur

#### La page ne se charge pas

**Solutions** :
1.  Vérifiez votre connexion Internet
2.  Actualisez la page (F5 ou Ctrl+R)
3.  Videz le cache de votre navigateur
4.  Essayez un autre navigateur (Chrome, Firefox, Edge)

---

### 9.2 Problèmes d'affichage

#### Les contacts ne s'affichent pas

**Solutions** :
1.  Vérifiez que vous êtes bien connecté (votre nom apparaît en haut à droite)
2.  Actualisez la page
3.  Vérifiez qu'un filtre de recherche n'est pas actif
4.  Si vous n'avez aucun contact, ajoutez-en un pour commencer

#### Les images ne s'affichent pas

**Solutions** :
1.  Vérifiez votre connexion Internet
2.  Actualisez la page
3.  L'image par défaut devrait s'afficher si aucune photo n'est définie

---

### 9.3 Problèmes d'ajout/modification

#### "Veuillez corriger les erreurs dans le formulaire"

**Solutions** :
1.  Regardez les champs avec une **bordure rouge**
2.  Lisez le **message d'erreur** sous chaque champ
3.  Corrigez les erreurs indiquées
4.  Les bordures deviennent **vertes** quand le champ est correct

#### L'upload de photo ne fonctionne pas

**Solutions** :
1.  Vérifiez que le fichier est au format **JPG** ou **PNG**
2.  Vérifiez que la taille est **inférieure à 2 MB**
3.  Réessayez avec une autre photo
4.  Si le problème persiste, continuez sans photo (ajoutez-la plus tard)

---

### 9.4 Problèmes généraux

#### Le bouton ne répond pas

**Solutions** :
1.  Attendez quelques secondes (un spinner peut s'afficher)
2.  Ne cliquez qu'**une seule fois**
3.  Si rien ne se passe après 10 secondes, actualisez la page

#### J'ai perdu mes données

**Vérifications** :
1.  Êtes-vous connecté avec le **bon compte** ?
2.  Avez-vous accidentellement **supprimé** les contacts ?
3.  Si vos données ont disparu, contactez votre administrateur

## 10.  Questions fréquentes (FAQ)

### Compte et connexion

**Q : Puis-je changer mon nom d'utilisateur ?**  
R : Non, le nom d'utilisateur est définitif. Choisissez-le bien lors de l'inscription.

**Q : Puis-je changer mon mot de passe ?**  
R : Pour le moment, cette fonctionnalité n'est pas disponible. Contactez votre administrateur.

**Q : Combien de temps reste ma session active ?**  
R : Votre session reste active tant que vous ne fermez pas le navigateur. Pour plus de sécurité, déconnectez-vous manuellement.

---

### Gestion des contacts

**Q : Combien de contacts puis-je créer ?**  
R : Il n'y a pas de limite technique. Vous pouvez créer autant de contacts que nécessaire.

**Q : Puis-je partager mes contacts avec un collègue ?**  
R : Non, chaque utilisateur a ses propres contacts privés. Cette fonctionnalité pourrait être ajoutée dans une future version.

**Q : Puis-je exporter mes contacts en CSV ou Excel ?**  
R : Cette fonctionnalité n'est pas encore disponible mais est prévue dans une prochaine version.

**Q : Les contacts supprimés peuvent-ils être récupérés ?**  
R : Non, la suppression est définitive. Soyez prudent avant de supprimer un contact.

---

### Fonctionnalités

**Q : Puis-je ajouter des champs personnalisés ?**  
R : Non, les champs sont prédéfinis. Si vous avez besoin d'informations supplémentaires, utilisez le champ "Note".

**Q : Y a-t-il des rappels automatiques pour les relances ?**  
R : Non, vous devez consulter manuellement vos contacts à relancer via la recherche avancée.

**Q : Puis-je trier mes contacts autrement que par ordre alphabétique ?**  
R : Pour le moment, seul le tri alphabétique par nom est disponible.

---

### Technique

**Q : Sur quels navigateurs fonctionne Mini CRM ?**  
R : Mini CRM fonctionne sur tous les navigateurs modernes : Chrome, Firefox, Edge, Safari.

**Q : Puis-je utiliser Mini CRM sur mon téléphone ?**  
R : Oui, l'application est **responsive** et s'adapte aux mobiles et tablettes.

**Q : Mes données sont-elles sauvegardées automatiquement ?**  
R : Oui, chaque action (ajout, modification, suppression) est immédiatement sauvegardée dans la base de données.

**Q : Où sont stockées mes données ?**  
R : Vos contacts sont stockés sur **Airtable** (service cloud sécurisé) et vos photos sur **Cloudinary**.
